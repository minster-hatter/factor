# factor
Some notes of Turing's Factor Principle.

# Background

Alan Turing worked on decrypting cryptographic messages (famously, enigma) at Bletchley Park during the Second World War. Some of his documents were declassified in 2015. One such document was [The Applications of Probability to Cryptography](https://arxiv.org/pdf/1505.04714). This paper has some discussion of updating beliefs about hypotheses in a fundamentally Bayesian manner. Given the technological limitations of the time, the paper simplifies matters in a way that would not be necessary today. Therefore, it makes sense to take the ideas (which were clearly extremely useful), and build on them using modern techniques.
